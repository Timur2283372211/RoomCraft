Інструкція

Щоб створити кімнату, використовуйте кнопки для додавання фігур та об'єктів.

Ви можете змінювати кольори, видаляти обрані об'єкти та зберігати проект.

Для малювання використовуйте кнопку "paint Mode".

Щоб додати текст, натисніть кнопку "Додати текст".

Для редагування тексту, натисніть кнопку "Редагувати текст".

Щоб закрити інструкцію, натисніть кнопку "Закрити"/escape.

Для збереження проекту натисніть кнопку "Зберегти проект".

Для видалення всіх об'єктів натисніть кнопку "delete All".

Для видалення обраних об'єктів натисніть кнопку "delete Selected".

Для додавання об'єктів, таких як диван, туалет або принтер, використовуйте відповідні кнопки.

Для малювання ліній, трикутників, зірок та інших фігур, використовуйте відповідні кнопки.

Для малювання точок, використовуйте кнопку "draw dot Mod".

Для завершення фігури, натисніть кнопку "Завершити фігуру".

Для малювання на полотні, використовуйте кнопку "paint Mode".

Для того щоб перейти в початкову позицію натисніть 0

Для того щоб переміщатися зажміть лкм(ліва клавіша миші) + ctrl

Для того щоб змінити масштаб покрутіть колесо миші

Редактор кімнат не доступний на малих пристроях

Будь ласка, відкрийте цей сайт з комп’ютера або пристрою з шириною екрана більше ніж 900 пікселів.
